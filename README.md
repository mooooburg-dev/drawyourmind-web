# Overview

2000년대 초반 개인 홈페이지가 유행하던 그때 그시절의 감성을 끄집어내다.  
당시에 홈페이지 좀 만든다는 사람들끼리 서로의 개인 홈페이지를 왕래하면서  
방명록에 글을 남기고 '축전'이라는 것을 만들어 전달하던 그 시절 추억에 잠겨 홈페이지를 구상하던 중,  
그 시절의 그 것과 지금의 그 것을 섞어보기로 했다.
